# Security Policy

## Supported Versions

RF Bridge is an actively developed project. Security fixes and updates are generally applied to the latest release version.

| Version | Supported |
|---------|-----------|
| Latest Release | ✅ |
| Older Releases | ⚠️ Best effort |

---

## Reporting a Vulnerability

If you discover a security issue or vulnerability, please report it responsibly.

Please avoid publicly disclosing exploit details immediately through GitHub Issues or social media. Instead:

- Open a GitHub issue with limited detail, or
- Contact the maintainer directly to coordinate a fix

Include as much relevant information as possible:
- RF Bridge version
- macOS version
- Steps to reproduce
- Logs or screenshots if applicable
- Hardware involved (TinySA model, adapters, etc.)

---

## Scope

Potential security concerns may include:
- Unsafe file handling
- Unexpected serial device behavior
- Dependency vulnerabilities
- Malicious or malformed device responses
- Permission or packaging issues in distributed app builds

---

## Responsible Disclosure

Reasonable efforts will be made to investigate and address legitimate reports in a timely manner.

This project is currently maintained as an independent/open-source effort, so response times may vary depending on availability.
