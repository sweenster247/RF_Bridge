# RF Bridge v1.9.4.1 — Overlay UI and Marker Color Patch

RF Bridge v1.9.4.1 is a small usability patch focused on the main window layout and marker customization.

## Changed

- Reduced the connection panel footprint so it no longer dominates the top of the app
- Added a dedicated capture overlay section beside the connection panel
- Added quick overlay show/hide checkboxes directly in the main window
- Expanded Mic Plot / Marker color choices

## Fixed

- Transient empty scan reads now retry quietly a few times before interrupting the user with a warning dialog

## Notes

This patch keeps the v1.9.4 profiles, overlays, Help/About menu, wiki, icon, and consolidated build workflow intact.
