# RF Bridge v1.9.1 — Mic Plot Label Fix

RF Bridge v1.9.1 is a small polish update for the new Mic Plot feature introduced in v1.9.

## Fixed

- Moved Mic Plot labels down into the visible graph area so marker names are easier to read
- Updated marker labels to show the user-defined name and frequency with MHz suffix
- Added subtle label background and border for better contrast against the RF graph

## Notes

This release does not change the Mic Plot data format or existing saved markers.
