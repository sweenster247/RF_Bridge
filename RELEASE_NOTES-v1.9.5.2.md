# RF Bridge v1.9.5.2 — Sidebar and Layout Polish

RF Bridge v1.9.5.2 is a small UI polish release focused on the left sidebar branding and main dashboard spacing.

## Changed

- Refined the left sidebar header with a larger clean RF Bridge logo asset.
- Moved the app name and “RF Spectrum Analyzer” subtitle beneath the logo for better readability.
- Widened the sidebar to prevent subtitle truncation.
- Reduced outer dashboard padding so the RF graph and main content use more of the existing window.
- Tightened the Capture Overlays and connected-device cards so the top row feels less crowded.

## Notes

This release preserves the confirmed-working v1.9.4.10 tinySA connection behavior and the recent v1.9.4.11/v1.9.5 graph axis/cursor stability fixes.
