# RF Bridge v1.9.4.12

RF Bridge v1.9.4.12 is a UI layout cleanup release based on the confirmed-working v1.9.4.10 tinySA connection path and the v1.9.4.11 graph/overlay stability fixes.

## Changed

- Added a left-side navigation rail for common app actions
- Moved connected-device controls to the top-right of the main window
- Recentered the Capture Overlays panel into the top section
- Matched the Capture Overlays panel height more closely with the connected-device panel
- Centered the empty-overlay state text for a cleaner layout

## Retained

- v1.9.4.10 tinySA startup timing behavior
- v1.9.4.11 axis/cursor stability fixes
- Capture overlay toggle crash fix
- Mic Plot / marker workflow
- Capture overlays and profile support

## Notes

This release focuses on visual polish and layout consistency. It does not intentionally change tinySA serial communication behavior.
